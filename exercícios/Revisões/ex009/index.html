Capítulo 08

Negrito

• Para criarmos negrito semântico, como esse, usamos a tag <strong></strong>

• Para criarmos negrito não semântico, como esse, usamos a tag <b></b>

Itálico

• Para criarmos um itálico semântico, como esse, usamos a tag <em></em>

• Para criarmos um itálico não semântico, como esse, usamos a tag <i></i>

Texto inserido

• Para criarmos um texto inserido semântico, como esse, usamos a tag <ins></ins>

• Para criarmos um texto inserido não semântico, como esse, usamos a tag <u></u>

Texto marcado

Para criarmos um texto marcado, como esse, usamos a tag <mark></mark> dentro do body ou dentro do head

Texto deletado

• Para criarmos um texto deletado, como esse, usamos a tag <del></del>

Texto sobrescrito

• Para criarmos um texto sobrescrito, como: x2 + 33, usamos a tag <sup></sup>

Texto subscrito

• Para criarmos um texto subscrito, como: CO2 H20, usamos a tag <sub></sub>

Texto Grande e Pequeno

• Para criarmos texto grande, como esse, usamos a tag <big></big> que todavia é uma tag que vai deixar de existir em breve.

• Para criarmos texto pequeno, como esse, usamos a tag <small></small>

OBS: Ao contrário de texto grande, texto pequeno é semântico, isso porque existem letras pequenas em contratos, o que ainda tem significado nessas condições de uso.

Capítulo 09

Código-Fonte

Para fazermos o nosso texto ter cada caractere o mesmo espaçamento nas laterais, como esse exemplo: como diz o amigo do meu amigo que não é meu amigo: o computador é um burro muito rápido. Usamos a tag <code></code>

• Para fazer código de programação de forma indentada usamos <pre><code></code></pre> respectivamente.

Citações simples

• Exemplo de citação simples: "O computador é um burro muito burro rápido"

OBS: Para isso usamos a tag <q></q>

Citações completas

• Exemplo de citação completa:

O computador é um burro muito burro rápido

OBS: Para isso usamos a tag <blockquote></blockquote>

Abreviações

• Para fazermos abreviações, como esses exemplos: HTML e CSS usamos a tag <abbr></abbr>
