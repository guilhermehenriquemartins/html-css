Itálico

Existem duas formas de fazer o Itálico:

O semântico, na qual usamos a tag <em> e </em> e o não semântico usando a tag <i> e </i>

Também existem duas formas de fazer o Negrito:

O negrito semântico, representado pela tag <strong> e </strong> e o negrito não semântico, representado pela tag <b> e </b>

Texto grande e pequeno

A tag de texto grande no HTML não é mais recomendável, porém para se fazer no HTML fazemos o seguinte:

Podemos fazer dessa forma, utilizando a tag <big> e </big>

Podemos fazer texto pequeno dessa forma, para isso utlizamos a tag <small> e </small> ao contrário de texto grande, esse tipo de tag para texto pequeno ainda continua sendo recomendável, isso porque ainda existem nos contratos partes de textos bem pequenos, o que é algo que ainda faz sentido.

Texto deletado

Para fazermos texto deletado, como dessa forma, utilizamos a tag <del>

Texto inserido

Para fazermos texto inserido existem duas formas:

Texto inserido semântico, para isso utilizamos a tag <ins> e </ins>

Texto inserido não semântico, usamos a tag <u> e </u>

Texto sobrescrito

Para texto sobrescrito utilizamos a tag <sup> e </sup>, que normalmente usamos para fins como: x2 + 35

Texto subscrito

Para texto subscrito utlizamos a tag <sub> e </sub>, que normalmente utilizamos para fins como: H20, CO2, etc
